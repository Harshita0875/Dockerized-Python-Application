# Dockerized-Python-Application
Dockerized Python 3.12 application that prints container Python version and current date-time.
#BUILD DOCKER IMAGE :
Using command : docker build -t <image-name?
In this project, docker build -t python-version-app

#RUN DOCKER IMAGE:
Using command : docker run <image-name>
In this project, docker run python-version-app
